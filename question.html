Question-1: What are the different ways to select an element in the DOM?

There are five ways to select element in dom.
1. getElementById()
2. getElementByClassName()
3. getElementByTagName()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML -> Get or sets HTML content inside an element, includes html
innerText -> Get or sets the visible text inside an element
textContent -> Get or sets all text content inside an element

Question-3: What is event delegation in the DOM?

It is powerful techniques. use when multiple element dynamically to simple event management. addEventListener() use it.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the dom when an event like a click happens on a child element moves on a parent element and other anchestor.

Question-5: How do you create, add, and remove elements using JavaScript?

i want to create element using JavaScript use: document.createElement();
i want to add element using JavaScript use: document.body.appendChild();
i want to remove element using JavaScript use: document.remove();
